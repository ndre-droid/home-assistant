Erste Einrichtung (einmalig)

🔌Geräte verbinden
1
Hue: Geräte-Tab → Bridge-IP eintragen (Hue-App → Einstellungen → Meine Hue-Systeme) → runden Knopf auf der Bridge drücken → Koppeln
2
Sonos: Geräte-Tab → Suchen — Beam & Era 100 erscheinen automatisch
3
LG TV: Name, IP & MAC eintragen (TV: Einstellungen → Netzwerk → WLAN → Erweitert) → TV hinzufügen → Koppeln → am TV bestätigen
4
Claude-Key: Einstellungen-Tab → API-Key von console.anthropic.com einfügen (5 $ Guthaben reicht lange)

Täglich nutzen

💬Automation per Sprache erstellen
Automationen-Tab → + → „Neue Automation“ → einfach beschreiben:
„Wenn ich den NFC-Tag berühre: alle Lichter aus, Sonos pausieren, TV aus.“
→ Los. Claude baut die Routine, du prüfst die Aktionen → Speichern.
▶️Automationen starten & schalten
▶ = sofort ausführen · Schalter = Trigger aktiv/inaktiv · Antippen = öffnen & bearbeiten
🎬Szene aufnehmen (Scene Sync)
1
Licht & Musik jetzt so einstellen, wie die Szene sein soll
2
+ → „Szene aus aktuellem Zustand aufnehmen“
3
Geräte anhaken, beim Beam optional Night-Mode → Name („Movie Night“) → speichern
Farbe, Helligkeit, Lautstärke & laufende Wiedergabe werden eingefroren.
📱Widget & NFC-Tag
Widget: Homescreen lange drücken → Widgets → HomeFlow → platzieren → Automation wählen. Ein Tipp startet sie.

NFC: Automation öffnen → Auslöser NFC → „NFC-Tag beschreiben“ → Tag an die Handy-Rückseite halten.
Tag antippen startet die Routine sofort — Bildschirm muss entsperrt sein (Android-Vorgabe).
🔴Alles aus
Geräte-Tab → „Alles aus“ oben rechts: alle Lampen, Sonos & TV gleichzeitig, in unter einer halben Sekunde.

Fortgeschritten

🌗Tag/Nacht-Varianten
Eine Automation kann sich tags/nachts unterscheiden (grün + Vögel / blau gedimmt + Wale). Bei Claude einfach mitbeschreiben oder im Editor „+ Tag/Nacht-Variante“. Zeiten (Standard 07:00 / 22:00) in den Einstellungen.
💡Geräte-Trigger („wenn Badlicht angeht“)
Auslöser Hue-Licht → Lampe wählen → „wird eingeschaltet“. Läuft über einen Hintergrunddienst.
Samsung-Pflicht: Einstellungen → Apps → HomeFlow → Akku → Nicht optimiert — sonst wird der Dienst beendet.
📺TV Bias-Light (ohne Sync-Box)
Einstellungen → TV Bias-Light aktivieren → Lightstrips anhaken. Je nach TV-Inhalt:
Netflix / Prime / HDMIwarmes dunkles Orange
YouTubeViolett
Live-TVkühles Blau
TV ausStrips aus
Ehrlich: Stimmung nach Content-Typ, Reaktion ~5 s. Frame-genaues Ambilight braucht HDMI-Capture-Hardware — gibt die TV-API nicht her.
🌍Unterwegs steuern (Tailscale)
1
Tailscale-App auf dem Handy installieren, anmelden
2
Tailscale auf einem Gerät zuhause, das durchläuft: tailscale up --advertise-routes=192.168.178.0/24
3
Route auf login.tailscale.com freigeben
VPN an = HomeFlow funktioniert unterwegs exakt wie zuhause.
🔊Sound-URLs (Vögel, Wale …)
Zeigt eine Aktion ⚠ Sound-URL fehlt: antippen, URL eintragen. Quellen: Internetradio-Naturgeräusch-Sender oder eigene MP3 vom Heimrechner (python -m http.server → http://PC-IP:8000/voegel.mp3).

Probleme & Lösungen

🛠️Wenn etwas nicht funktioniert
Hue koppelt nichtBridge-Knopf drücken, innerhalb 30 s koppeln. IP & WLAN prüfen
Sonos nicht gefundenGleiches WLAN (kein Gäste-WLAN)? Nochmal suchen
TV-Kopplung scheitertTV muss an sein · IP prüfen · „LG Connect Apps“ am TV erlauben
TV geht nicht ANMAC eingetragen? „Einschalten über WLAN“ am TV aktiv? Nach Stromlos-Zustand prinzipiell unmöglich
Trigger/Bias stoppt nach StundenAkku-Optimierung ausschalten (s. o.) — Benachrichtigung „HomeFlow aktiv“ muss sichtbar sein
NFC reagiert nichtBildschirm entsperren · NFC an · Tag an obere Rückseite
Claude-FehlerAPI-Key & Guthaben prüfen · Internet nötig (nur beim Erstellen)
Spotify-Szene unvollständigSonos-Grenze: Quelle ja, exakte Queue-Position nicht immer
Era 100 + Night-Mode = FehlerNormal — Night-Mode nur Beam/Arc
Neue Version bauenDateien ins Repo (Ordner HomeFlow/) → Actions → Artifact laden → APK drüberinstallieren
